<h2>Python3 random password generator with GUI built on Tkinter</h2>

Dependencies:
<ul>
    <li>tkinter</li>
    <li>pyperclip</li>
</ul>

Open the folder and run the script:

<code> python3 password_generator_GUI.py</code>
